# Выпускной проект: платформа для публикации статей

Этот репозиторий содержит выпускной проект Акулинина Владислава Сергеевича по программе профессиональной переподготовки «Веб-разработчик» (256 академических часов). Приложение демонстрирует освоение современного стека веб-разработки и реализует полный цикл работы с контентом: создание, редактирование, хранение и публикация статей.

## Описание программы переподготовки

Курс «Веб-разработчик» ориентирован на подготовку специалистов для эффективного создания веб-сайтов, интерактивных приложений и информационных ресурсов. На программе изучаются алгоритмы и программное обеспечение для разработки веб-проектов, инструменты интернет-коммуникаций, процессы сопровождения информационных систем, а также современные инструментальные средства обработки исходного кода.

## Технологический стек

- React 19 и TypeScript 5 — декларативный UI, статическая типизация;
- Vite 7 — быстрая разработка и сборка;
- Tailwind CSS 4 — адаптивная стилизация компонентов;
- TanStack Router и TanStack Query — маршрутизация и работа с данными;
- Supabase — облачная платформа хранения и синхронизации контента;
- @uiw/react-md-editor — редактор и предпросмотр Markdown;
- Vitest и Testing Library — модульное и интеграционное тестирование;
- ESLint и Prettier — контроль качества и единого стиля кода.

## Быстрый старт

```bash
pnpm install
pnpm dev
```

Приложение запустится c Vite на порту 3000.

### Переменные окружения

Создайте `.env` на основе `.env.example` и заполните:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_KEY`
- `SUPABASE_DB_URL` — строка подключения Postgres с SSL.

### Настройка базы данных

```bash
pnpm db:setup
```

Скрипт создаст таблицу `articles`, используя SQL из `scripts/supebase/setupArticles.ts` и подключение `SUPABASE_DB_URL`.

## Основные команды

- `pnpm dev` — запуск режима разработки;
- `pnpm build` — продакшн-сборка (Vite + tsc);
- `pnpm test` — запуск тестов (Vitest);
- `pnpm lint` — проверка ESLint;
- `pnpm format` — проверка форматирования;
- `pnpm check` — автоформатирование и линтинг.

## Структура проекта

```
src/
  components/     UI-компоненты и общие блоки
  routes/         файловая маршрутизация TanStack Router
  service/        работа с Supabase и бизнес-логика
  shared/         вспомогательные utils, адаптеры, типы
```

Главная страница (`src/routes/index.tsx`) содержит описание проекта и быстрый доступ к созданию новой статьи. Редактор статей размещён в `src/routes/e/articles.$articleId.tsx`, а публичные страницы отображения материалов — в `src/routes/articles`.

## Тестирование и качество

- Тесты: Vitest + Testing Library (`pnpm test`);
- Lint: ESLint на базе `@tanstack/eslint-config`;
- Форматирование: Prettier (`pnpm format`).
